# Ngeblog

This is a blog platform.

## Installation

1. Install
```bash
composer require ngeblog/app
```
2. Create database and seeting your `.env`
3. Run php artisaan
```bash
php artisan ngeblog:install
```
4. Run php artisan
```bash
php artisan storage:link
```
4. Run npm
```bash
npm run install && npm run production
```

## Feature

1. Article
2. Comment
3. Tag
4. Search
